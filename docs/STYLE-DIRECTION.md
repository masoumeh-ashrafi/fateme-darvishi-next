# Style Direction

## Selected Direction

Luxury Editorial Beauty + Premium PMU Specialist + Transformation Proof

## Design Mix

- 70% Luxury Editorial Beauty
- 20% Premium PMU Specialist
- 10% Transformation First

## Brand Feeling

The website should feel elegant, warm, premium, feminine, trustworthy, and conversion-focused.

It should not feel like a generic beauty template or a simple blog.

## Visual Direction

- Soft luxury
- Editorial layout
- Large clean imagery
- Warm neutral background
- Strong spacing
- Clear reservation CTA
- Trust-building sections
- SEO-friendly content structure

## Color Palette

- Background: `#FAF4EF`
- Surface: `#FFFFFF`
- Primary Text: `#2B2024`
- Secondary Text: `#7A646A`
- Accent: `#B47A6B`
- Soft Accent: `#E8C7BD`
- Gold: `#C9A46A`
- Border: `#EADBD5`

## Design Rules

- Do not redesign randomly.
- Do not remove useful old content.
- Do not make the site look like a generic template.
- Prioritize SEO, trust, and booking conversion.
- Every section must have a purpose.
- Every service should have enough content for search intent.
- Portfolio and testimonials must support user trust.
- Blog content should support service pages through internal linking.

## Main Pages

- Home
- Services
- Single Service Pages
- Portfolio
- Blog
- Single Blog Post
- About
- Contact / Booking
- FAQ

## Priority

The first goal is not visual polish.

The first goal is:

1. Content migration
2. SEO structure
3. Trust-building content
4. Conversion path
5. UI polish
